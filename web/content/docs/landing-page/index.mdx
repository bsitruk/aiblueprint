Use this page to choose a landing-page direction before asking an agent to
redesign a public homepage. Pick one lead reference and at most two supporting
references. Blending more than three usually makes the result generic.

The screenshots were captured from live websites on July 4, 2026 using a
1440x1000 desktop viewport. The set is intentionally split into mobile-first
references and supporting desktop/product references, because the original list
mixed both.

## How to choose

- For a mobile app landing page, start from the first group.
- For native app polish, device mockups, or App Store conversion, use the second
  group as support.
- For motion, premium dark product proof, or developer-tool polish, borrow only
  the specific idea from the supporting product and desktop group.
