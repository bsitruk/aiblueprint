`agents pro` manages AIBlueprint Premium. Premium adds extra agents, skills,
scripts, settings, and tooling on top of the free defaults.

Start with `setup` for a first installation. It asks for your access token when
none is saved yet.

```bash
aiblueprint agents pro setup
```

## Subcommands

| Command | Description |
| ------- | ----------- |
| [`agents pro activate [token]`](/commands/pro-activate) | Activate Premium with an access token. |
| [`agents pro status`](/commands/pro-status) | Check whether a Premium token is saved. |
| [`agents pro setup`](/commands/pro-setup) | Install Premium and its required tooling. |
| [`agents pro update`](/commands/pro-update) | Replace the local Premium configuration with the latest complete version. |
| [`agents pro sync`](/commands/sync) | Review changes and selectively import Premium updates. |

## Update or sync?

Use [`update`](/commands/pro-update) when you want the complete latest Premium
configuration. Use [`sync`](/commands/sync) when you want to inspect the diff,
choose which additions or modifications to import, and create a backup first.

## Custom folders

Commands that read or install configuration inherit the folder options from
`agents`. Place these options before `pro`:

```bash
aiblueprint agents --folder ~/my-config pro setup
```

See each command page for its exact behavior and examples.

Get a Premium token at [mlv.sh/claude-cli](https://mlv.sh/claude-cli).
